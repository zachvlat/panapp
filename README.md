
# Green Portal

Green Portal is an experimental mobile application focused on providing the latest news and updates about the Panathinaikos team. The app aggregates news from various sources and displays the team's roster.

## Features

- **News Aggregation**: Fetches and displays news from multiple sources.
- **Football Roster**: Displays the current football roster with player details.
- **Custom Drawer Navigation**: Provides an easy-to-use navigation menu.

## Building

1. Clone the repository:
   ```sh
   git clone https://github.com/zachvlat/panapp.git
   cd panapp

2. Install dependencies:
   ```sh
   npm install
   ```

3. Run the app:
   ```sh
   npx expo start
   ```
